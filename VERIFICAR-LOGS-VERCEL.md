# Como Verificar Logs do Vercel

## 1. Acesse os logs em tempo real:
https://vercel.com/lojaecompack-hashs-projects/tiny-expedicao/logs

## 2. Ou veja os logs do último deployment:
https://vercel.com/lojaecompack-hashs-projects/tiny-expedicao/deployments

Clique no último deployment e vá em "Functions" → Procure por erros relacionados a `/api/users`

## 3. O que procurar:
- Erros de conexão com banco de dados
- Erros de Prisma Client
- Erros de timeout
- Mensagens de erro específicas
